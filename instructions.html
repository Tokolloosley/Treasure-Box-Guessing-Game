Treasure Box Guessing Game

Game Instructions

Game Goal

The goal of the game is to open boxes and collect rewards while avoiding traps. Some boxes contain treasure, while others contain penalties.

How to Play

  1. Enter your player name on the launcher page.
  2. Select the difficulty level.
  3. Select the grid size.
  4. Choose a theme such as Treasure, Space Crates, or Mystery Doors.
  5. Select optional features such as hints, shield, or double reward mode.
  6. Click Open Game Window.
  7. In the game window, click Start Game.
  8. Click unopened boxes one at a time.
  9. Reward boxes increase your score, while trap boxes reduce lives or score.
  10. Clear all safe boxes or survive with the best score possible.

Difficulty Levels

  • Easy: fewer traps and more safe boxes
  • Medium: balanced number of rewards and traps
  • Hard: more traps and higher risk

Buttons in the Game Window

  • Start Game starts a new game
  • Reveal Safe Hint shows one safe unopened box if available
  • Save Session stores the current game state
  • Load Session restores a saved game
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Opening reward boxes adds points
  • Opening trap boxes reduces points or lives
  • Double reward mode increases points from reward boxes
  • If enabled, a shield can protect you from one trap
